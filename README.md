# 📊 Data Professional Survey Breakdown — Power BI Dashboard

This project presents an interactive Power BI dashboard that visualizes and analyzes survey responses from 617 data professionals worldwide. It explores trends in salaries, programming language preferences, career entry difficulty, and job satisfaction across various data roles and geographies.

## 📁 Project Files

| File                                 | Description                                                       |
| ------------------------------------ | ----------------------------------------------------------------- |
| `Data_Industry_Survey_Analysis.pbix` | Main Power BI report file containing all visuals and calculations |

## 📌 Dashboard Overview

The single-page interactive dashboard includes:

* **Average Salary by Job Title** – Bar chart comparing salaries across roles
* **Favourite Programming Language** – Column chart showing language preferences
* **Country of Survey Takers** – Treemap for geographic distribution
* **Difficulty Breaking into Data** – Donut chart representing career entry difficulty
* **Happiness with Salary** – Gauge showing compensation satisfaction
* **Happiness with Work/Life Balance** – Gauge for work/life satisfaction
* **Count of Survey Takers & Average Age** – KPI cards summarizing respondents

## 🛠️ Tools & Skills Used

* **Power BI Desktop (v2026.02)** for dashboard development
* **Power Query** for data cleaning, transformation, and deriving average salaries
* **DAX** for calculated measures such as averages, counts, and satisfaction scores
* **Data Visualization** with bar charts, column charts, treemaps, donut charts, gauges, and KPI cards

## 📊 Dataset

* Real-world survey collected by [Alex Freberg (Alex the Analyst)](https://www.youtube.com/@AlexTheAnalyst)
* **617 records** across **28 columns** covering: job titles, salary ranges, countries, favourite programming languages, career entry difficulty, satisfaction scores, and age

## 🔍 Key Insights

* **Data Scientists** earn the highest average salaries, followed by Data Engineers and Data Architects
* **Python** is the most popular programming language
* The **United States** leads in average salaries, while **India** has the lowest
* Most respondents rated entering the data field as **moderate difficulty**
* Work/life balance satisfaction scores are generally higher than salary satisfaction

## 🚀 How to View

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Open `Data_Industry_Survey_Analysis.pbix`
3. Explore the dashboard with slicers and cross-filtering for dynamic insights
